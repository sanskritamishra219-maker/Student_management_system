🎓 College ERP System

A web-based College ERP (Enterprise Resource Planning) System developed to streamline academic and administrative operations within a college. The application provides dedicated portals for Admins, Faculty Members, and Students, enabling efficient management of attendance, marks, timetables, and user records through a centralized platform.

---

✨ Features

👨‍💼 Admin Module

* Manage student and faculty records
* Add, update, and delete users
* Search and filter records
* Export data to CSV format
* Monitor academic information

👨‍🏫 Faculty Module

* Mark and manage student attendance
* Enter and update marks
* View teaching schedules and timetables
* Access student academic records

👨‍🎓 Student Module

* View attendance and attendance percentage
* Check subject-wise marks
* Access personal timetable
* Monitor academic performance
